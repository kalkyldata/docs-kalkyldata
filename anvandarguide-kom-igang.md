# 1. Kom igång

Det här avsnittet hjälper dig att snabbt förstå produkten och komma igång på några minuter.

## Introduktion / översikt
KalkylData är till för att förenkla ditt arbete med kalkyler och relaterad dokumentation i en strukturerad process.

## Installation / registrering
Följ installationsguiden i dokumentationen för att konfigurera systemet i din miljö.

## Snabbstart (5–10 min)
1. Logga in eller skapa konto.
2. Öppna första arbetsytan/projektet.
3. Kör ett första grundflöde enligt instruktionerna.
4. Verifiera att resultat visas korrekt.
