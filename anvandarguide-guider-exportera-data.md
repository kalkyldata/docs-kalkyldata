# Så här exporterar du data

## Mål
Exportera relevant data för delning, uppföljning eller vidare analys.

## Förutsättningar
- Du har åtkomst till den vy eller rapport som ska exporteras.
- Du har rätt behörighet för export.

## Steg
1. Öppna vyn/rapporten med data du vill exportera.
2. Klicka på **Exportera**.
3. Välj format (t.ex. Excel/CSV/PDF om tillgängligt).
4. Bekräfta exporten.
5. Kontrollera den nedladdade filen.

## Resultat
Data finns i vald filtyp och kan användas utanför systemet.
